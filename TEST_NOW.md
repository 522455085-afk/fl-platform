## 1. 注册昵称修复（昨晚最大的 bug）

### 1A. 全新邮箱注册新账号
- [ ] 打开 `/login` → 切「注册」
- [ ] 用一个**全新没用过的邮箱**注册，用户名输入「**测试昵称**」
- [ ] 收到验证码 → 输入验证 → 进入主页
- [ ] **看左下角小卡片**，用户名是「**测试昵称**」 → 通过 ✅
- [ ] 如果显示成邮箱前缀 → 失败 ❌（截图给 Cascade）

### 1B. 老账号自动修复
- [ ] 退出登录
- [ ] 用之前注册过的、用户名错误成邮箱前缀的**老账号**登录
- [ ] **看左下角**，用户名应该自动改回正确的（注册时输入过什么就是什么）
- [ ] 如果还是邮箱前缀 → 进 F12 控制台贴错误信息 → 给 Cascade

> **原理**：`ensureProfile` 会检查现有行的 username 是否等于邮箱前缀，是就 update 成 pendingRegisterUsername。但如果你注册时就一直没填 username，repair 不会触发（因为本来就是 email 前缀）。
